# SunBloom Geometry Calculator

SunBloom Geometry Calculator adalah website kalkulator geometri berbasis web yang dibuat untuk membantu pengguna menghitung berbagai kebutuhan geometri secara lebih mudah, cepat, dan interaktif.

Website ini dibuat dengan tampilan modern, responsif, dan ringan sehingga dapat digunakan langsung melalui browser tanpa perlu instalasi tambahan.

## Live Demo

Website dapat diakses langsung melalui link berikut:

https://calkulator-geometri.netlify.app/

## Deskripsi Proyek

Proyek ini merupakan kalkulator geometri online yang dirancang untuk membantu pengguna dalam melakukan perhitungan rumus-rumus geometri. Aplikasi ini cocok digunakan sebagai media belajar, latihan, maupun demonstrasi proyek web interaktif.

Dengan tampilan yang sederhana namun menarik, pengguna dapat menggunakan kalkulator ini untuk memahami konsep perhitungan geometri secara lebih praktis.

## Fitur Utama

- Kalkulator geometri berbasis web
- Tampilan modern dan interaktif
- Dapat diakses langsung melalui browser
- Responsif untuk berbagai ukuran layar
- Cocok digunakan di laptop, tablet, maupun smartphone
- Tidak memerlukan instalasi aplikasi tambahan
- Ringan dan mudah digunakan
- Cocok untuk pembelajaran matematika dan geometri

## Teknologi yang Digunakan

- HTML
- CSS
- JavaScript
- Netlify untuk deployment

## Tujuan Proyek

Tujuan dari proyek ini adalah membuat sebuah website kalkulator geometri yang mudah digunakan, memiliki tampilan menarik, dan dapat membantu pengguna dalam melakukan perhitungan geometri secara praktis.

Selain itu, proyek ini juga dibuat sebagai bagian dari pengembangan portofolio web development, khususnya dalam pembuatan aplikasi interaktif berbasis HTML, CSS, dan JavaScript.
